Translation: (x, y) → (x + h, y + k) — slides without rotating or flipping
Question 1 — Translate a Triangle
Triangle DEF was translated to D′E′F′. If D = (−4, −1) and D′ = (1, −3), identify the translation rule.
D(−4,−1)→D′(1,−3): x change = 1−(−4) = +5, y change = −3−(−1) = −2. Rule: (x + 5, y − 2) ✓
Question 2
Translate G(−3, 7) using (x + 6, y − 5). Find G′.
(−3+6, 7−5) = (3,2) = G′(3, 2) ✓
Question 3
After rule (x − 4, y + 7), point H′ is at (2, 3). Find the original H.
Work backwards: H = H′ − (−4,+7) = (2+4, 3−7) = H(6, −4) ✓
Question 4
Triangle with vertices (−2,0),(2,0),(0,3) is translated (x+3, y−1). What is the new position of (0,3)?
(0+3, 3−1) = (3, 2) ✓
Question 5
A figure is translated (x+2, y−3), then translated again (x−5, y+1). What single rule equals both translations combined?
Add the h values: 2+(−5)=−3. Add the k values: −3+1=−2. Combined rule: (x−3, y−2) ✓
Question 6
Point M(5, −2) is translated 3 units left and 6 units up. What are M′'s coordinates?
3 left = −3, 6 up = +6. M(5,−2) → (5−3, −2+6) = M′(2, 4) ✓

x-axis reflection: (x,y)→(x,−y) | y-axis reflection: (x,y)→(−x,y)
Question 1 — Reflect a Rectangle
Rectangle WXYZ was reflected. What line of reflection was used?
W(1,3)→W′(1,−3): x stayed, y changed sign → reflection over x-axis. Rule: (x,y)→(x,−y). ✓
Question 2
Reflect N(−6, 2) over the y-axis.
y-axis: (x,y)→(−x,y). N(−6,2) → (6,2) = N′(6, 2) ✓
Question 3
Reflect P(4, −8) over the x-axis.
x-axis: (x,y)→(x,−y). P(4,−8) → (4,8) = P′(4, 8) ✓
Question 4
A point Q′ after reflection over the y-axis is at (5, 3). Find original Q.
y-axis: (x,y)→(−x,y). Q′(5,3) = (−x,y) → x = −5, y = 3. Original: Q(−5, 3) ✓
Question 5
Triangle A(2,1), B(6,1), C(4,5) reflected over x-axis. Find B′.
x-axis reflection: (x,y)→(x,−y). B(6,1) → B′(6, −1) ✓
Question 6
What do reflections over the x-axis and y-axis have in common?
All reflections are rigid transformations → congruent figures. The x and y axes each flip a different coordinate, but both preserve size and shape. ✓

Question 1 — 270° CCW Rotation
Triangle ABC was rotated 270° CCW. If A = (2, 1), find A′.
270° CCW: (x,y)→(y,−x). A(2,1) → (1,−2) = A′(1, −2) ✓
Question 2
Rotate V(−3, 5) by 90° CCW.
90°CCW: (x,y)→(−y,x). V(−3,5) → (−5,−3) = V′(−5, −3) ✓
Question 3
Rotate W(6, −2) by 180°.
180°: (x,y)→(−x,−y). W(6,−2) → (−6,2) = W′(−6, 2) ✓
Question 4
Rotate X(−4, −1) by 90° CW (= 270° CCW).
90°CW = 270°CCW: (x,y)→(y,−x). X(−4,−1) → (−1,−(−4)) = (−1,4) = X′(−1, 4) ✓
Question 5
Which rotation maps (3, 5) to (−5, 3)?
90°CCW: (3,5)→(−5,3). Check: (x,y)→(−y,x) = (−5,3) ✓. 90° CCW
Question 6
A figure is rotated 90°CCW three times. What single rotation gives the same result?
90°×3 = 270° total → 270° CCW. (Same as 90°CW). ✓

Dilation: (x,y)→(kx,ky) | k>1=enlargement | 0<k<1=reduction | Similar figures
Question 1 — Dilation k = 2
Square ABCD was dilated with k = 2. If A = (1, 1), find A′.
k=2: A(1,1) → (2×1, 2×1) = A′(2, 2) ✓. Each coordinate multiplied by 2.
Question 2
Dilate Y(9, −12) with k = 1/3.
k=1/3: (9×1/3, −12×1/3) = (3, −4) = Y′(3, −4) ✓. Reduction.
Question 3
The image Z′ is at (−20, 8). If k = 4, find original Z.
Z = Z′/k = (−20/4, 8/4) = (−5, 2) = Z(−5, 2) ✓
Question 4
Rectangle with perimeter 20 is dilated k = 3. New perimeter?
Perimeter scales by |k|: 20 × 3 = 60 ✓. Area would scale by k² = 9.
Question 5
Triangle area = 16 sq units. After dilation k = ½, what is the new area?
Area scales by k²: 16 × (1/2)² = 16 × 1/4 = 4 sq units ✓
Question 6
A dilation maps every point farther from the origin. This is an:
Points moving farther from origin = coordinates getting larger = enlargement (k > 1). ✓

Mixed Review: These questions combine all transformation types.
Question 1
Which transformation changes the SIZE of a figure?
Only dilation changes size. Translations, reflections, and rotations are all rigid — they preserve size and shape. ✓
Question 2
Point A(3, −2): apply rotation 90°CCW, then reflection over x-axis. Find A″.
Step 1 rotate 90°CCW: A(3,−2)→(−(−2),3)=(2,3). Step 2 reflect over x-axis: (2,3)→(2,−3). (2, −3) ✓
Question 3
Figure M is translated, reflected, then rotated. Is the result congruent or similar to M?
All three (translation, reflection, rotation) are rigid. No dilation → congruent. ✓
Question 4
What is the result of reflecting (a, b) first over the y-axis, then over the x-axis?
Over y-axis: (a,b)→(−a,b). Over x-axis: (−a,b)→(−a,−b). Final: (−a, −b). Same as 180° rotation! ✓
Question 5
A 10-unit segment is dilated k = 2.5. What is the new length?
Length scales by |k|: 10 × 2.5 = 25 units ✓
Question 6
Translate P(0, 5) by (x + 3, y − 8), then dilate k = 2. Find P″.
Step 1 translate: P(0,5)→P′(3,−3). Step 2 dilate k=2: P′(3,−3)→P″(6,−6). (6, −6) ✓
Question 7
Figures X and Y are congruent. Which transformation was NOT used?
Congruent figures = same size and shape. A dilation changes size → figures would only be similar. Dilation was NOT used. ✓
Question 8
What transformation maps EVERY point (x, y) to (−x, −y)?
(x,y)→(−x,−y) is the rule for 180° rotation about the origin. It's also equivalent to reflecting over both axes. ✓

🔍 Common Mistakes

Can you spot what went wrong?

Rotate (3,−2) by 90° CCW
✗ Student: Student writes: (−2, 3)
✓ Fix: WRONG! Rule is (−y, x). y=−2, so −y = 2. Correct: (2, 3)
💡 Be careful with negatives in rotation rules. −(−2) = +2.
Dilate (4,6) by k=½
✗ Student: Student writes: (2, 8)
✓ Fix: WRONG! ½ × 4 = 2 ✓ but ½ × 6 = 3, not 8. Correct: (2, 3)
💡 Multiply BOTH coordinates by the scale factor.

⚡ Key Tips for This Unit

✅ Translation: (x+a, y+b) — slide. CONGRUENT.
✅ Reflection: x-axis → negate y. y-axis → negate x. CONGRUENT.
✅ Rotation 90° CCW: (−y, x). 180°: (−x,−y). 270° CCW: (y,−x). CONGRUENT.
✅ Dilation: (kx, ky). SIMILAR (not congruent unless k=1).
