6.1 Introduction to Functions

NC Standards: NC.8.F.1 — Understand that a function is a rule that assigns to each input exactly one output.

What is a Function?

A function is a relation (set of ordered pairs) in which every input (x) has only one corresponding output (y) value. Functions can be represented in different ways.

Functions as Ordered Pairs, Tables, and Mapping

For these three representations, the input and output values are listed separately.

  • Ordered Pairs represent the related inputs and outputs as a list of coordinate pairs.
  • Mappings use arrows to indicate the relationship between input and output values.
  • Tables display related input and output pairs in rows and columns.
⚠️ Important: If any input (x) has more than one output (y), it is NOT a function!

Functions as Graphs

Functions can also be represented by graphs of these values.

Discrete
Individual, separate points — values are counted (e.g., number of tickets sold).
Continuous
An unbroken line or curve — values flow without interruption (e.g., temperature over time).
⚠️ Vertical Line Test: If a vertical line can be drawn anywhere on a graph and crosses it in two or more places, the relation is NOT a function.

Functions as Equations

Functions can also be represented by equations. The equation is sometimes referred to as the Function Rule.

Example: y = ½x + 4

Sometimes, instead of using y to represent the output, equations will use Function Notation and replace y with f(x) — read as "f of x".

Standard form:
y = 2x + 4
Function Notation:
f(x) = 2x + 4
🔑 Using Function Notation: f(3) means "find the output when x = 3." If f(x) = 2x + 4, then f(3) = 2(3) + 4 = 10.
⚠️ Important: If an equation has y in the denominator or uses y², it is NOT a function. Vertical lines (e.g., x = 6) are NOT functions. Horizontal lines (e.g., y = 7) ARE functions.

Examples of NON-functions: 2x + 3y = 4/y   or   x² + y² = 9

Evaluating Functions with Equations

To evaluate a function, substitute the given input into the equation to determine the corresponding output. Always use parentheses when substituting!

Example: For f(x) = 3x − 3, find f(2)

f(2) = 3(2) − 3
f(2) = 6 − 3
f(2) = 3 ✓

Evaluating Functions with Graphs

For graphs, find the input (x) value on the x-axis, then move directly up or down to locate where it intersects the graph. The corresponding y-value is your output.

Example: Given f(x), evaluate f(−3)

Find −3 on the x-axis. Move up to where the graph intersects at (−3, 2).
f(−3) = 2 ✓

Domain and Range

Every function has a domain and a range. We list these values in { } braces, or describe them using an inequality.

DOMAIN
All Input (x) values
"What x-values are allowed?"
RANGE
All Output (y) values
"What y-values result?"

Increasing and Decreasing Behavior

When looking at a graph, describe the function behavior moving from left to right.

DirectionGraph BehaviorWhat It Means
Increasing ↗Goes UP left to rightOutput (y) gets larger as input (x) increases
Decreasing ↘Goes DOWN left to rightOutput (y) gets smaller as input (x) increases
Constant →Stays flat / horizontalOutput (y) does not change
🔑 Note: A single function can have multiple behaviors — decreasing, then constant, then decreasing again, then increasing.

6.2 Linear Functions and Slope

NC Standards: NC.8.F.3 — Interpret y = mx + b as defining a linear function. NC.8.F.4 — Determine rate of change and initial value.

There are many different types of functions, but the most common is the Linear Function. A Linear Function has a constant rate of change. When graphed, it produces a straight line. We use the term slope to describe the rate of change.

Slope

Slope is the ratio of the vertical change (Δy) to the horizontal change (Δx) — often called "rise over run."

slope = Δy / Δx = (y₂ − y₁) / (x₂ − x₁)

Four Types of Slope

TypeGraph DirectionKey WordExample
PositiveGoes up left to right ↗Increasingm = 2
NegativeGoes down left to right ↘Decreasingm = −3
ZeroPerfectly horizontal →Constantm = 0
UndefinedPerfectly vertical ↕Not a functionx = k

Slope Formula — Given Two Points

Example: Find the slope through (2, 4) and (10, 2)

Step 1 — Label: Point 1: (x₁=2, y₁=4)   Point 2: (x₂=10, y₂=2)
Step 2 — Substitute: (2 − 4) / (10 − 2) = −2 / 8
Step 3 — Simplify: −2/8 = −1/4 ✓

Finding Slope from a Graph

Identify two clear points on the graph. Starting from the left point, count how many spaces you move up or down (Δy) and how many spaces you move right (Δx).

Slope-Intercept Form

y = mx + b
VariableNameMeaningHow to Identify
mSlopeRate of changeThe number multiplied by x
bY-InterceptStarting value (when x = 0)The constant at the end

Linear vs. Non-Linear Functions

A linear function has a constant rate of change — equal steps in x always produce equal steps in y. A non-linear function does NOT have a constant rate of change.

✓ Linear: y = 2x + 1
x: 0→1→2→3 | y: 1→3→5→7
Rate ALWAYS +2 → LINEAR
✗ Non-Linear: y = x² + 1
x: 0→1→2→3 | y: 1→2→5→10
Rate NOT constant → NON-LINEAR
🔑 Note: Other non-linear examples: y = x² (parabola/quadratic), y = 2ˣ (exponential). Their graphs are CURVES, not straight lines.

6.3 Writing Linear Equations from Tables

NC Standards: NC.8.F.4 — Construct a function to model a linear relationship; determine rate of change from tables.

We can determine whether a table represents a linear function and then write it in slope-intercept form. To check for linearity, calculate Δy ÷ Δx between consecutive rows. If the slope is always the same, the relationship is linear.

⚠️ Important: Always check SLOPE (Δy ÷ Δx), not just Δy alone — x-values may not increase by 1 each time!

Steps to Write the Equation from a Table

Steps:

1. Find the slope: Pick any two points. slope = (y₂ − y₁) ÷ (x₂ − x₁)
2. Find the y-intercept: Use the row where x = 0, or substitute slope + a known point into y = mx + b and solve for b.
3. Write the equation in slope-intercept form: y = mx + b

Worked Example

Given: x → y:   −2→−4,   −1→−1,   0→2,   1→5,   2→8,   3→11

Step 1 — Find slope: Use (0, 2) and (1, 5): m = (5−2) ÷ (1−0) = 3
Step 2 — Find y-intercept: When x = 0, y = 2 → b = 2
Step 3 — Write equation: y = 3x + 2 ✓

6.4 Linear Functions in Real-World Context

NC Standards: NC.8.F.4 — Construct a function to model a linear relationship; interpret rate of change and initial value in context.

A linear equation models a relationship between two quantities where the change is constant (a steady rate of change).

Writing Equations from Context — Keywords

m (Slope = Rate of Change)
"per," "each," "every," "for each additional"
A consistent change for every unit increase in x
b (Y-Intercept = Starting Value)
"initial value," "starting amount," "deposit"
"base fee," "flat rate," "one-time fee"

Worked Examples

Example 1: A plumber charges $80 after 1 hour and $140 after 3 hours.

Step 1 — Label points: (1, 80) and (3, 140)
Step 2 — Find slope: m = (140 − 80) ÷ (3 − 1) = 60 ÷ 2 = 30
Step 3 — Find b: 80 = 30(1) + b → b = 50
Equation: y = 30x + 50 ✓

Example 2: A taxi charges $3 flat fee plus $2.50 per mile. How much for 7 miles?

Identify m: $2.50 per mile → m = 2.50
Identify b: $3 flat fee → b = 3
Equation: y = 2.50x + 3
7-mile ride: y = 2.50(7) + 3 = 17.50 + 3 = $20.50 ✓

Interpreting Slope and Y-Intercept

Practice: The value of a car after x years is y = −1500x + 24000

What does slope (−1500) mean? The car loses $1,500 in value every year.
What does y-intercept (24000) mean? The car was worth $24,000 when new (year 0).
After 5 years: y = −1500(5) + 24000 = $16,500 ✓

6.5 Comparing Functions

NC Standards: NC.8.F.2 — Compare properties of two functions each represented in a different way.

Linear functions can be represented in four ways. When comparing, convert both to slope-intercept form (y = mx + b) so you can directly compare slopes and y-intercepts.

🔑 Strategy: Before answering any comparison question, find slope and y-intercept for BOTH functions first, then answer the specific question.

Practice Problem 1

Compare: Function A (table: 0→5, 2→11, 4→17, 6→23) vs Function B (y = 4x + 8)

Function A: m = (11−5)÷(2−0) = 3, b = 5
Function B: m = 4, b = 8 (read directly)
Higher rate of change: Function B (4 > 3)   |   Higher initial value: Function B (8 > 5)

Practice Problem 2

Compare slopes: Function A (graph through (0,6) and (3,0)) vs Function B (points: x=0→y=2, x=4→y=10)

Function A: m = (0−6)÷(3−0) = −6/3 = −2
Function B: m = (10−2)÷(4−0) = 8/4 = 2
Function A has a negative slope, Function B has a positive slope.

Practice Problem 3 — Cell Phone Plans

Plan A: y = 0.05x + 20 vs Plan B: $30 monthly + $0.02/min

Plan A: m = 0.05, b = $20   |   Plan B: m = 0.02, b = $30
At 500 min: A = 0.05(500)+20 = $45   |   B = 0.02(500)+30 = $40
Plan B has higher initial cost. Plan B is cheaper at 500 min ($40 vs $45) ✓

Practice Problem 4 — Savings Accounts

Account A: Starts $300, adds $25/week. Account B: (0,$400), (2,$440), (4,$480)

Account A: y = 25x + 300
Account B: m = (440−400)÷(2−0) = $20/wk, b = $400 → y = 20x + 400
After 10 weeks: A = 25(10)+300 = $550   |   B = 20(10)+400 = $600
Account B has more after 10 weeks ($600 > $550) ✓

Practice Problem 5 — Candle

A graph shows candle height passing through (0, 12) and (4, 8).

Y-intercept: At x=0, the candle is 12 inches — its full starting height.
Slope: m = (8−12)÷(4−0) = −4/4 = −1 → candle burns 1 inch per hour
Equation: y = −x + 12 ✓

Practice Problem 6 — Delivery

Store 1: y = 2x + 5   Store 2: y = 3x + 2   You live 10 miles away.

Store 1: 2(10) + 5 = $25   |   Store 2: 3(10) + 2 = $32
Store 1 is cheaper for 10 miles ($25 vs $32) ✓

Problem 1 — Swimmer

A swimmer swims at a steady pace to the far wall, rests, then swims back faster.

Phase 1: Distance increases steadily → straight line going UP
Phase 2: Resting at wall → FLAT horizontal line
Phase 3: Swimming back faster → straight line going DOWN (steeper than phase 1)
⚠️ Key insight: When she swims BACK, the distance from start DECREASES — the line goes DOWN, not up. Faster = steeper.

Problem 2 — Read the Graph

A distance-time graph: steep up (0–3 min), flat (3–6 min), gradual up (6–10 min)

0 to 3 min: Steep upward line → moving away quickly (fast, steady speed)
3 to 6 min: Flat / horizontal → stopped, not moving
6 to 10 min: Gradual upward line → moving away slowly (slow, steady speed)

Problem 3 — Boiling Water

A pot of water: heats slowly, then rapidly, stays at boiling, then cools gradually.

Heating slowly: Shallow upward curve
Heating rapidly: Steep upward curve
Boiling: Flat horizontal line (100°C)
Cooling: Downward curve (decreasing)

6.7 Systems of Equations

NC Standards: NC.8.EE.8a, 8b, 8c — Analyze and solve pairs of simultaneous linear equations; interpret solutions in real-world context.

A solution for a system of linear equations is the coordinate (x, y) that is true for both functions. There are a few ways to find the solution.

Graphing Method

Graph both equations on the same coordinate plane and look for where they intersect.

One Solution
Lines intersect at one point. Different slopes.
No Solution
Lines are parallel. Same slope, different b.
Infinite Solutions
Lines are identical. Same slope, same b.

Inspection — Quick Check

What You NoticeResultExample
Different slopesONE solutiony = 3x+2 and y = −x+6
Same slope, different bNO solutiony = 3x+2 and y = 3x−5
Identical equationsINFINITE solutionsy = 3x+2 and 2y = 6x+4

Substitution Method

Example 1: y = 4x − 1 and y = 2x − 5

Set equal: 4x − 1 = 2x − 5
−2x from both: 2x − 1 = −5
+1 to both: 2x = −4 → x = −2
Solve for y: y = 4(−2) − 1 = −9
Solution: (−2, −9) ✓

Example 2: y = x + 9 and 3x + 8y = −5

Substitute: 3x + 8(x + 9) = −5
3x + 8x + 72 = −5 → 11x + 72 = −5
11x = −77 → x = −7
Solve for y: y = −7 + 9 = 2
Solution: (−7, 2) ✓

Word Problems

Problem 1: Cell Phone Plans — A: $30/mo + $0.10/text   B: $40 flat. When equal?

Equations: A: y = 0.10x + 30   B: y = 40
Solve: 0.10x + 30 = 40 → 0.10x = 10 → x = 100
Equal at 100 texts. Beyond 100, Company B is cheaper. ✓

Problem 2: Fundraiser — Kids $3, adults $5. $250 total, 70 tickets sold.

Equations: c + a = 70   and   3c + 5a = 250
Substitute: c = 70 − a → 3(70−a) + 5a = 250 → 210 − 3a + 5a = 250 → 2a = 40 → a = 20
c = 70 − 20 = 50
50 children's + 20 adult tickets ✓ Check: 3(50)+5(20)=$250 ✓

Problem 3: Car Rentals — X: $20 + $0.30/mi   Y: $35 + $0.15/mi

Solve: 0.30x + 20 = 0.15x + 35 → 0.15x = 15 → x = 100 miles
At 50 miles: X = $35   Y = $42.50
Equal at 100 mi. For 50 mi, Company X is cheaper ($35). Over 100 mi, Y is cheaper. ✓
⚠️ Common Error: When solving a system, make sure to find BOTH x and y. Many students solve for x and forget to substitute back to find y!

🔍 Spot the Error: Find slope from (3, 7) and (6, 1)
✗ Student: m = (6 − 3)/(1 − 7) = 3/(−6) = −1/2
✓ Fix: The student swapped rise and run! It should be (y₂−y₁)/(x₂−x₁) = (1−7)/(6−3) = −6/3 = −2

💡 Remember

Rise (y-change) goes on TOP, Run (x-change) goes on BOTTOM. Never swap them!
🔍 Spot the Error: Is this a function?
✗ Student says: "It's a function because each output has one input."
✓ Fix: The rule is each INPUT must have exactly one output. Here, input 2 maps to BOTH 5 and 7 — not a function!
🔍 Spot the Error: System of Equations
✗ Student: Solve y = 3x + 1 and y = 3x − 4. Student writes: 3x + 1 = 3x − 4 → 1 = −4 → "x = −5"
✓ Fix: 1 = −4 is a FALSE statement (variables cancelled). This means NO SOLUTION — the lines are parallel (same slope of 3, different y-intercepts).

💡 Remember

Same slope + different y-intercept = parallel lines = no solution. Same slope + same y-intercept = same line = infinite solutions.
