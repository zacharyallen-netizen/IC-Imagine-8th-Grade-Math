🔄 Unit 8 Practice — Transformations

Translations · Reflections · Rotations · Dilations · Congruence & Similarity · NC.8.G.1–4

Score: 0|Answered: 0 / 12
Reflection: x-axis → (x, −y) · y-axis → (−x, y) · y=x → (y, x)

90° CW: (x,y)→(y,−x) · 90° CCW: (x,y)→(−y,x) · 180°: (x,y)→(−x,−y) · Dilation: (kx, ky)

💡 Tip: Translations, reflections & rotations are rigid motions (preserve size → congruent). Dilations change size (→ similar).
➡️ Translations (NC.8.G.1)
Problem 8A
A point is translated by the rule (x + 3, y − 2). If the original point is (1, 5), what is the image?
Add 3 to x: 1 + 3 = 4. Subtract 2 from y: 5 − 2 = 3. Image: (4, 3)
🪞 Reflections (NC.8.G.3)
Problem 8B
What is the image of (−2, 4) reflected over the x-axis?
Reflect over x-axis: keep x, negate y → (x, −y). So (−2, 4) → (−2, −4)
Problem 8C
What is the image of (3, −1) reflected over the y-axis?
Reflect over y-axis: negate x, keep y → (−x, y). So (3, −1) → (−3, −1)
Problem 8K
What is the image of (−4, 2) reflected over the line y = x?
Reflect over y = x: swap coordinates → (x, y) → (y, x). So (−4, 2) → (2, −4)
🔄 Rotations (NC.8.G.3)
Problem 8D
Rotate (2, 5) 90° clockwise about the origin. What is the image?
90° clockwise: (x, y) → (y, −x). So (2, 5) → (5, −2)
Problem 8E
Rotate (−3, 1) 180° about the origin. What is the image?
180° rotation: (x, y) → (−x, −y). So (−3, 1) → (3, −1)
Problem 8L
Rotate (1, 0) 90° counter-clockwise about the origin. What is the image?
90° CCW: (x, y) → (−y, x). So (1, 0) → (0, 1)
🔍 Dilations (NC.8.G.3)
Problem 8F
A triangle is dilated by scale factor k = 2, centered at the origin. Point A(3, 4) maps to:
Dilation: multiply each coordinate by k. (3×2, 4×2) = (6, 8)
Problem 8G
A figure is dilated by scale factor k = ½. The original side length is 10 cm. What is the image side length?
Multiply by scale factor: 10 × ½ = 5 cm. Scale factor < 1 makes it smaller.
🧩 Congruence & Similarity (NC.8.G.2, NC.8.G.4)
Problem 8H
Which transformation changes the SIZE of a figure?
Only dilations change size! Translations, reflections, and rotations preserve size (they are rigid motions).
Problem 8I
A figure is translated, then reflected. The result is:
Translations and reflections are rigid motions — they preserve both size and shape → congruent.
Problem 8J
Triangle A has sides 3, 4, 5. Triangle B has sides 6, 8, 10. What transformation maps A to B?
Each side doubled: 3→6, 4→8, 5→10. Scale factor k = 2 dilation.

🔍 Common Mistakes

Can you spot what went wrong?

Rotate (3,−2) by 90° CCW
✗ Student: Student writes: (−2, 3)
✓ Fix: WRONG! Rule is (−y, x). y=−2, so −y = 2. Correct: (2, 3)
💡 Be careful with negatives in rotation rules. −(−2) = +2.
Dilate (4,6) by k=½
✗ Student: Student writes: (2, 8)
✓ Fix: WRONG! ½ × 4 = 2 ✓ but ½ × 6 = 3, not 8. Correct: (2, 3)
💡 Multiply BOTH coordinates by the scale factor.

⚡ Key Tips for This Unit

✅ Translation: (x+a, y+b) — slide. CONGRUENT.
✅ Reflection: x-axis → negate y. y-axis → negate x. CONGRUENT.
✅ Rotation 90° CCW: (−y, x). 180°: (−x,−y). 270° CCW: (y,−x). CONGRUENT.
✅ Dilation: (kx, ky). SIMILAR (not congruent unless k=1).
