🔄 Unit 8: Transformations

Translations · Reflections · Rotations · Dilations · Congruence & Similarity · NC.8.G.1–4

8.1 Transformation Vocabulary
📌 Transformation: A rule that moves, flips, turns, or resizes every point of a figure in a predictable way.
📌 Pre-image: The original figure BEFORE the transformation.
📌 Image: The new figure AFTER the transformation. We label image vertices with a prime symbol: A → A′ (read "A prime").
📌 Rigid Transformation (Isometry): A transformation that preserves BOTH size AND shape. The pre-image and image are congruent. Types: translation, reflection, rotation.
📌 Non-Rigid Transformation: A transformation that changes the SIZE but preserves the shape. The pre-image and image are similar. Type: dilation.
8.2 Translations (Slides)
📌 Translation: Every point slides the same distance in the same direction. The shape is NOT rotated or flipped — it just moves.

The Rule

Translation Rule: (x, y) → (x + h, y + k)
• h = horizontal shift (positive = right, negative = left)
• k = vertical shift (positive = up, negative = down)

Example: Translate triangle with vertices A(1,3), B(4,3), C(4,6) by (−3, −2)

Rule: (x, y) → (x + (−3), y + (−2)) = (x − 3, y − 2)
A(1, 3) → A′(1−3, 3−2) = A′(−2, 1)
B(4, 3) → B′(4−3, 3−2) = B′(1, 1)
C(4, 6) → C′(4−3, 6−2) = C′(1, 4) ✓ — Shape is congruent; shifted 3 left and 2 down

✓ Check: Point P(−2, 5) is translated by (x+4, y−3). What are the new coordinates?

(−2+4, 5−3) = (2, 2) ✓
8.3 Reflections (Flips)
📌 Reflection: Every point flips across a line of reflection (the mirror line). Each point is the same distance from the mirror line on the opposite side.

Reflection Rules — NC.8.G.3 Focus

📌 NC Standard: 8th grade focuses on reflections across the x-axis and y-axis only.
Line of ReflectionRuleExample: Start (3, 4)What Changes
x-axis (horizontal axis)(x, y) → (x, −y)(3, 4) → (3, −4)y-coordinate flips sign
y-axis (vertical axis)(x, y) → (−x, y)(3, 4) → (−3, 4)x-coordinate flips sign
📌 Memory trick: Reflecting over the x-axis? The x-coordinate stays, y flips. Reflecting over the y-axis? The y stays, x flips. The coordinate that matches the axis name is the one that stays.

Example: Reflect triangle A(2,1), B(5,1), C(5,4) over the line y = x

Rule for y = x: (x, y) → (y, x) — swap the coordinates!
A(2, 1) → A′(1, 2)
B(5, 1) → B′(1, 5)
C(5, 4) → C′(4, 5) ✓ — x and y coordinates are simply swapped

✓ Check: Point Q(−4, 7) is reflected over the x-axis. What are the new coordinates?

Reflect over x-axis: (x, y) → (x, −y). Q(−4, 7) → (−4, −7) ✓
8.4 Rotations (Turns)
📌 Rotation: Every point turns a fixed angle around a center of rotation (usually the origin). CCW = counter-clockwise. CW = clockwise.
💡 Key Fact: 90° CW is the same result as 270° CCW. 90° CCW is the same as 270° CW.

Rotation Rules About the Origin

RotationRuleExample: Start (3, 4)
90° CCW (or 270° CW)(x, y) → (−y, x)(3, 4) → (−4, 3)
180° (CCW or CW)(x, y) → (−x, −y)(3, 4) → (−3, −4)
270° CCW (or 90° CW)(x, y) → (y, −x)(3, 4) → (4, −3)
360° (full rotation)(x, y) → (x, y)(3, 4) → (3, 4)

Example: Rotate rectangle with vertices at (1,1), (4,1), (4,3), (1,3) by 90° CCW

Rule: (x, y) → (−y, x)
(1, 1) → (−1, 1)
(4, 1) → (−1, 4)
(4, 3) → (−3, 4)
(1, 3) → (−3, 1) ✓ — The rectangle rotates one quarter-turn CCW around the origin

✓ Check: Point M(5, −2) is rotated 180° about the origin. What are the new coordinates?

180°: (x, y) → (−x, −y). M(5, −2) → (−5, 2) ✓
8.5 Dilations (Size Changes)
📌 Dilation: Every point is stretched away from (or compressed toward) a center point using a scale factor k. Dilations produce SIMILAR figures — same shape, different size.
📌 Scale Factor (k):
• k > 1 → Enlargement (bigger)
• 0 < k < 1 → Reduction (smaller)
• k = 1 → same size (congruent)
• k < 0 → dilated AND reflected through center

Dilation Rule (center at origin)

Dilation Rule: (x, y) → (kx, ky)
Multiply BOTH coordinates by the scale factor k.

Example: Enlarge — k = 3

A(2, 1) → A′(6, 3)
B(4, 1) → B′(12, 3)
C(3, 3) → C′(9, 9) ✓

Example: Reduce — k = ½

A(6, 4) → A′(3, 2)
B(8, 0) → B′(4, 0)
C(2, 2) → C′(1, 1) ✓
⚠️ What changes in a dilation: Side lengths multiply by |k|. Area multiplies by k². Perimeter multiplies by |k|. What stays the same: Angle measures. Shape. Orientation (unless k is negative).

✓ Check: Triangle A(2,4), B(6,4), C(4,8) is dilated with k = ½. What is the image of B?

(kx, ky) = (½×6, ½×4) = (3, 2) ✓

Composite Transformations

📌 Composite Transformation: Two or more transformations applied one after another. Order matters! — doing A then B gives a different result than doing B then A.

Example: Translate then Reflect

Start: Triangle A(1,1), B(3,1), C(2,3)
Step 1 — Translate right 2, down 3: (x+2, y−3)
A(1,1) → A′(3,−2), B(3,1) → B′(5,−2), C(2,3) → C′(4,0)
Step 2 — Reflect over the y-axis: (x,y) → (−x, y)
A′(3,−2) → A″(−3,−2), B′(5,−2) → B″(−5,−2), C′(4,0) → C″(−4,0) ✓

Example: Rotate then Dilate (produces SIMILAR figures)

Start: P(2, 4)
Step 1 — Rotate 90° CCW: (x,y) → (−y, x) → P′(−4, 2)
Step 2 — Dilate k = 3: (x,y) → (3x, 3y) → P″(−12, 6)
Final result: P″(−12, 6). Since a dilation was used, P and P″ are SIMILAR (not congruent). ✓
⚠️ Congruence Rule: If ALL transformations in a sequence are rigid (translation, reflection, rotation), the final image is congruent to the original. If even ONE dilation appears, the images are only similar.

✓ Check: Figure X is reflected, then rotated 90°, then translated. Is the final image congruent or similar to X?

Reflection, rotation, and translation are all rigid — no size change → figures are congruent ✓

📝 Worked Example

Composite: Translate (2,5) by (x−3, y+1), then reflect over x-axis

Step 1: Apply translation (x−3, y+1): (2−3, 5+1) = (−1, 6)

Step 2: Reflect over x-axis (x, −y): (−1, −6)

Step 3: Final image: (2, 5) → (−1, 6) → (−1, −6) ✓

🔍 Spot the Error: Reflect (4, −3) over the x-axis
✗ Student: (4, −3) → (4, −(−3)) = (4, 3) ✓
✓ Fix: This one is actually correct! Over x-axis: negate y. −(−3) = 3.

💡 Remember

Not every problem has an error! Being able to verify correct work is just as important.
🔍 Spot the Error: Rotate (3, −2) by 90° CCW
✗ Student: 90° CCW: (x,y) → (−y, x). So (3, −2) → (−2, 3)
✓ Fix: y = −2, so −y = −(−2) = 2, not −2. Correct: (2, 3)

💡 Remember

Be careful with double negatives! −(−2) = +2.
🔍 Spot the Error: Dilate (6, 4) by scale factor ½
✗ Student: (6, 4) → (½·6, ½·4) = (3, 8)
✓ Fix: ½ × 4 = 2, not 8. Student multiplied 4 by 2 instead of ½. Correct: (3, 2)

💡 Remember

Scale factor ½ makes things SMALLER. If your answer is bigger, something went wrong.

🔍 Spot the Error: Reflect (4, −3) over the x-axis

✗ Student: (4, −3) → (−4, −3)
✓ Fix: Over the x-axis, the y-value flips, not x! Rule: (x, y) → (x, −y). Correct: (4, 3)
💡 x-axis reflection = y flips | y-axis reflection = x flips. The OTHER coordinate changes!

🔍 Spot the Error: Rotate (3, −2) by 90° CCW

✗ Student: (3, −2) → (−2, 3) — just swapped x and y
✓ Fix: 90° CCW rule is (x, y) → (−y, x). So (3, −2) → (−(−2), 3) = (2, 3). The negative of −2 is positive 2!
💡 For 90° CCW: swap the coordinates, then negate the NEW x-value (the one that used to be y).

🔍 Spot the Error: Congruence vs Similarity

✗ Student says: "A dilation produces a congruent figure because it's a transformation."
✓ Fix: Dilations change SIZE, so they produce similar (same shape, different size) figures — NOT congruent. Only translations, reflections, and rotations produce congruent figures (same shape AND size).
