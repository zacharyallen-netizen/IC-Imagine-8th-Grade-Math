Unit 5: The Pythagorean Theorem

a² + b² = c² · Missing Sides · Word Problems · Coordinate Distance · NC.8.G.7 · NC.8.G.8

📋 NC Standards Addressed

  • NC.8.G.6 – Explain the Pythagorean Theorem and its converse.
  • NC.8.G.7 – Apply the Pythagorean Theorem to determine unknown side lengths in right triangles in real-world and mathematical problems.
  • NC.8.G.8 – Apply the Pythagorean Theorem to find the distance between two points in a coordinate system.

5.1 The Pythagorean Theorem

🔑 THE THEOREM: In a RIGHT triangle, the sum of the squares of the two LEGS equals the square of the HYPOTENUSE.

a² + b² = c²

a, b = legs (shorter sides)    c = hypotenuse (longest side, across from right angle)
⚠️ IMPORTANT: The Pythagorean Theorem ONLY works on RIGHT triangles (triangles with a 90° angle)!

5.2 Finding the Hypotenuse

🔑 WHEN TO USE: When you know BOTH LEGS and need to find the HYPOTENUSE.

Example: Legs are 6 and 8. Find the hypotenuse.

Step 1: Write the formula: a² + b² = c²
Step 2: Substitute: 6² + 8² = c²
Step 3: Square the numbers: 36 + 64 = c²
Step 4: Add: 100 = c²
Step 5: Take square root: √100 = c
Answer: c = 10 ✓

5.3 Finding a Missing Leg

🔑 WHEN TO USE: When you know ONE LEG and the HYPOTENUSE, and need to find the OTHER LEG.

Example: One leg is 5, hypotenuse is 13. Find the other leg.

Step 1: Write the formula: a² + b² = c²
Step 2: Substitute what you know: 5² + b² = 13²
Step 3: Square the numbers: 25 + b² = 169
Step 4: Subtract 25 from both sides: b² = 169 − 25
Step 5: Simplify: b² = 144
Step 6: Take square root: b = √144
Answer: b = 12 ✓

Practice Problems — Solve for a Missing Leg

⚠️ REMEMBER: When finding a LEG, you SUBTRACT from the hypotenuse squared. When finding the HYPOTENUSE, you ADD the two legs squared.

5.4 Converse of the Pythagorean Theorem

🔑 THE CONVERSE: Given three side lengths a ≤ b ≤ c, plug them into the formula and check: does a² + b² = c²?
If yes → right triangle. If no → not a right triangle.
Always test the two SMALLER sides against the LARGEST.

5.5 Word Problems

🔑 STRATEGY: (1) Read carefully · (2) Draw/label a diagram · (3) Identify legs vs hypotenuse · (4) Set up a² + b² = c² · (5) Solve step by step

Problem 1 — Ladder Against a Wall

A ladder 15 feet long leans against a wall. The base is 9 feet from the wall. How high does it reach?

Setup: 9² + h² = 15²
81 + h² = 225
h² = 225 − 81 = 144
h = √144 = 12 ft ✓

Problem 2 — Wheelchair Ramp

A ramp is 10 ft long and rises 6 ft vertically. What is the horizontal length?

Setup: a² + 6² = 10²
a² + 36 = 100
a² = 100 − 36 = 64
a = √64 = 8 ft ✓

Problem 3 — Baseball Diamond

A baseball diamond has 90 ft sides. What is the distance from home plate to second base (diagonal)?

Setup: 90² + 90² = d²
8100 + 8100 = 16200
d = √16200 ≈ 127 ft ✓

Problem 4 — TV Screen

A TV is 36 inches wide and 27 inches tall. Find the diagonal.

Setup: 36² + 27² = d²
1296 + 729 = 2025
d = √2025 = 45 in ✓

Problem 5 — Hiking Trail

A hiker walks 16 km east then 12 km north. Straight-line distance from start?

Setup: 16² + 12² = d²
256 + 144 = 400
d = √400 = 20 km ✓

Problem 6 — Flagpole Wire

A flagpole is 24 ft tall. A 26 ft wire runs from the top to a stake. How far is the stake from the base?

Setup: a² + 24² = 26²
a² + 576 = 676
a² = 676 − 576 = 100
a = √100 = 10 ft ✓

Problem 7 — Boat Trip

A boat travels 30 km east then 40 km north. Straight-line distance to the island?

Setup: 30² + 40² = d²
900 + 1600 = 2500
d = √2500 = 50 km ✓

Problem 8 — Garden Diagonal

A rectangular garden is 40 m wide and 30 m tall. How long is the diagonal path?

Setup: 40² + 30² = d²
1600 + 900 = 2500
d = √2500 = 50 m ✓

5.6 Distance on the Coordinate Plane

📐 HOW IT WORKS: Connect the two points to form a right triangle. The horizontal distance is Δx = |x₂ − x₁| and the vertical distance is Δy = |y₂ − y₁|. Then use a² + b² = c² where a = Δx, b = Δy, and c = distance.
⚠️ NO DISTANCE FORMULA NEEDED! You already know how to do this — just use the Pythagorean Theorem! Draw the right triangle, find Δx and Δy, then solve.

Worked Examples

Problem 1: Distance from (0, −2) to (6, 2)

Δx: |6 − 0| = 6    Δy: |2 − (−2)| = 4
Setup: 6² + 4² = d² → 36 + 16 = 52
d = √52 = 2√13 ≈ 7.21 ✓

Problem 2: Distance from (0, −2) to (2, 2)

Δx: |2 − 0| = 2    Δy: |2 − (−2)| = 4
Setup: 2² + 4² = d² → 4 + 16 = 20
d = √20 = 2√5 ≈ 4.47 ✓

Problem 3: Distance from (−3, 1) to (5, 7)

Δx: |5 − (−3)| = 8    Δy: |7 − 1| = 6
Setup: 8² + 6² = d² → 64 + 36 = 100
d = √100 = 10 ✓

Problem 4: Distance from (0, 0) to (3, 4)

Δx: |3 − 0| = 3    Δy: |4 − 0| = 4
Setup: 3² + 4² = d² → 9 + 16 = 25
d = √25 = 5 ✓ (Classic 3-4-5 triple!)

Problem 5: Distance from (−1, −1) to (5, 7)

Δx: |5 − (−1)| = 6    Δy: |7 − (−1)| = 8
Setup: 6² + 8² = d² → 36 + 64 = 100
d = √100 = 10 ✓

Problem 6: Distance from (−4, 3) to (8, −2)

Δx: |8 − (−4)| = 12    Δy: |−2 − 3| = 5
Setup: 12² + 5² = d² → 144 + 25 = 169
d = √169 = 13 ✓ (5-12-13 triple!)
⚠️ WATCH YOUR SIGNS: When subtracting negative coordinates, remember: subtracting a negative means ADDING!
Example: 5 − (−3) = 5 + 3 = 8, NOT 5 − 3 = 2

📝 Review: Worked Examples & Error Analysis

📝 Ladder: 15 ft ladder, base 9 ft from wall. How high?

Step 1: Ladder = hypotenuse (c = 15), base = leg (b = 9), height = leg (a = ?)
Step 2: Formula: a² + b² = c²
Step 3: Substitute: a² + 81 = 225
Step 4: Subtract 81: a² = 144 (finding a LEG = subtract)
Step 5: Square root: a = √144 = 12 ft ✓

🔍 Spot the Error: Find c: a = 6, b = 8

✗ Student work: 6² + 8² = c² → 36 + 64 = c² → 100 = c² → c = 50
✓ Fix: c² = 100, so c = √100 = 10, NOT 50! Always take the square root at the end.
💡 After adding a² + b², you still need to SQUARE ROOT the result to find c.

🔍 Spot the Error: Find missing leg: c = 13, a = 5

✗ Student work: 5² + b² = 13² → 25 + b² = 169 → b² = 169 + 25 = 194
✓ Fix: When finding a LEG, SUBTRACT: b² = 169 − 25 = 144, so b = 12
💡 Hypotenuse → ADD legs. Missing leg → SUBTRACT from hypotenuse squared.
